Terms and Definitions!

Terms Definitions
cd Command used to change directories
pwd Command that tells you where you currently are working
mkdir Command that creates a new directory
nano Command that allows you to write in or create a file
cp -r Command that copies something recursively
rm -fr Command that forcefully removes something
sudo Command that temporarily grants super-user priveleges
